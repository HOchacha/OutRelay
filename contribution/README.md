# Contributing

Thanks for considering a contribution. This directory has everything
you need to make a change land cleanly.

- [`CONTRIBUTING.md`](CONTRIBUTING.md) — how we expect a change to be
  proposed, reviewed, and merged.
- [`dev-loop.md`](dev-loop.md) — what to install, how to run tests,
  how to iterate on the controller without a cluster.
- [`code-style.md`](code-style.md) — the Go conventions enforced by
  `make` (gofmt, golangci-lint, gosec) plus the unwritten ones we
  care about.
- [`commits-and-prs.md`](commits-and-prs.md) — how to write commits
  and pull requests so review is fast.
- [`releasing.md`](releasing.md) — what happens when a tag is pushed.

If this is your first contribution, start with
[`CONTRIBUTING.md`](CONTRIBUTING.md).
